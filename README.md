# Bevyroids

A version of the game [Asteroids](https://en.wikipedia.org/wiki/Asteroids_(video_game)) powered
by Rust and [Bevy](https://bevyengine.org/).

A WASM version can be played online [here](https://reu.github.io/bevyroids/).
